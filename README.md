# Z Morse Code – Translator & Tool 📡

**Z Morse Code** is an intuitive, bidirectional translator for Morse code on Android. It was the **first-ever application released under the Zentix brand**.

## 🌟 Key Features

- **🔄 Bidirectional Translation**: Instant Text-to-Morse and Morse-to-Text conversion.
- **🔊 Audio Playback**: Listen to your translated Morse code message as a series of dots and dashes.
- **📚 Reference Chart**: Accurate international Morse code A-Z and 0-9 reference guide built-in.
- **📝 History Tracker**: Quickly recall and copy your previous translations.
- **📲 Social Share**: Share your translated Morse signals directly to messaging apps.
